```
4. 认知价值：帮助用户保住多少隐性知识？无法量化，但肯定值

年付¥120 的逻辑：
1. 相当于¥10/月，比月付便宜33%
2. 绑定用户一年，降低流失率
3. ¥120 = 一顿火锅 = 一整年的认知积累
```

**为什么不免费提供AI功能？**
```
AI调用有成本
• OpenAI/Claude API按token计费
• 一键周报生成约消耗0.01-0.05美元/次
• 如果完全免费，用户滥用会导致成本失控

我们相信：
• 为价值付费是长期关系的基础
• 月付¥15是对AI服务的合理尊重
• 免费试用可以，但长期使用理应为AI价值买单
```
