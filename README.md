# CompilerAI 编译器

**把每日笔记编译成结构化知识。**

记录零散的想法，编译器帮你提炼、关联、归档。

## 核心功能

- **编译真相**：上线区存放已整理的笔记，下线区暂存灵感碎片，每日自动整理归档
- **AI 辅助**：内置编译指令，一键清洗杂乱笔记，支持豆包/DeepSeek 双引擎
- **反向链接**：笔记间自动关联，提及即连接，无需手动维护
- **关系图谱**：可视化笔记网络，点击跳转，自由缩放拖拽
- **本地优先**：数据存在浏览器 IndexedDB，完全可控
