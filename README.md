# 编译器 - 职场知识库 MVP

> "你是你唯一的算法" —— 公司买的是8小时结果，认知源码只属于你

一个专注于职场知识管理的本地笔记应用，基于 Markdown 编辑，支持 AI 辅助和双向链接。

## ✨ 核心特性

### 基础功能
- **Markdown 编辑器** - 实时预览，支持 GFM 语法
- **标签系统** - 多维度标签分类管理
- **全文搜索** - 快速检索标题、内容、标签
- **AI 辅助** - 智能清洗文本、生成标签
- **周报生成** - 从笔记自动生成周报

### 🚀 核心优化 (v2.0)

#### UI优化
1. **Markdown 工具栏** - 快捷按钮：加粗、斜体、标题、列表、代码块、链接、引用等
2. **深色/浅色主题** - 一键切换，支持跟随系统
3. **快捷键帮助面板** - 按 `?` 键查看所有快捷键

#### 功能增强
4. **笔记模板** - 5种内置模板：会议纪要、日报、周报、复盘、问题记录
5. **快捷命令菜单** - 输入 `/` 触发命令菜单，快速插入 Markdown
6. **多格式导出** - 支持导出为 Markdown、HTML、PDF
7. **收藏夹** - 笔记收藏功能，侧边栏快速访问

#### 高级功能
8. **笔记双向链接** - 使用 `[[笔记标题]]` 语法创建链接，自动追踪引用关系
9. **高级搜索筛选** - 日期范围、标签筛选、多维度排序
10. **历史版本** - 每次保存自动记录版本，支持恢复到历史版本
11. **标签管理增强** - 分类颜色、删除确认、自动补全
12. **加载状态优化** - 骨架屏、AI 进度动画、保存状态指示
13. **保存状态指示器** - 实时显示：已保存 / 保存中 / 未保存更改

## 🤖 AI 功能配置

### 启用 AI 功能

1. **注册火山引擎账号**
   - 访问 [火山引擎控制台](https://console.volcengine.com)
   - 完成实名认证

2. **创建 API Key**
   - 进入 [API Key 管理页面](https://console.volcengine.com/ark/region:ark+cn-beijing/apiKey)
   - 点击「创建 API Key」并复制

3. **配置环境变量**
   ```bash
   # 复制示例配置
   cp .env.example .env
   
   # 编辑 .env 文件，填入你的 API Key
   VITE_ARK_API_KEY=你的API密钥
   ```

4. **重启应用**
   - 配置完成后，重新启动开发服务器
   - AI 功能会自动启用

### AI 功能说明

- **文本清洗**: 将情绪化的职场表达转换为专业、客观的语言
- **智能标签**: 根据内容自动提取相关标签
- **周报生成**: 从笔记内容自动生成结构化周报
- **洞察提炼**: 从内容中提炼核心结论和关键洞察

## 📦 技术栈

- **前端框架**: React 18 + TypeScript
- **构建工具**: Vite
- **样式**: Tailwind CSS
- **数据存储**: IndexedDB (本地)
- **Markdown**: react-markdown + remark-gfm
- **数据库**: idb (IndexedDB 封装)
- **AI 模型**: 豆包 Doubao-Seed-2.0-lite (火山引擎)

## 💡 使用技巧

### 创建笔记
1. 点击侧边栏「新建笔记」或使用 `Ctrl+N`
2. 选择模板或从空白笔记开始
3. 编写 Markdown 内容
4. 使用工具栏快速格式化
5. 按 `Ctrl+S` 保存

### 双向链接
在笔记中使用 `[[其他笔记标题]]` 语法创建链接：
```markdown
参考 [[项目复盘]] 了解详情
```

### 使用模板
新建笔记时可以选择模板，快速创建结构化文档：
- 📋 会议纪要
- 📅 日报
- 📊 周报
- 🔄 复盘
- ❓ 问题记录

### 快捷命令
在编辑器中输入 `/` 可以快速插入：
- 各种标题级别
- 列表和待办事项
- 代码块
- 表格
- 日期等

## 🔒 数据安全

- 所有数据存储在本地 IndexedDB
- 不上传任何数据到服务器
- 关闭浏览器后数据仍然保留
- 支持清除浏览器数据导出备份
