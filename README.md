# CompilerAI 编译器

**把每日笔记编译成结构化知识。**

> 「你是你唯一的算法」

---

## 解决什么问题

- 写周报时想不起干了啥，翻遍聊天记录也凑不够三行
- 离职前疯狂整理文档，恨不得把脑子里的东西都倒出来
- 接手新项目时一脸懵，前任的笔记东一块西一块
- 记了一堆笔记，回头找的时候完全不记得存在哪

编译器帮你把脑子里的碎片整理清楚，关键时刻能找到、想用时能调出来。

---

## 核心功能

**情绪清洗** — 内置 200+ 情绪词映射，一键转成专业表述，保留原意  
**一键周报** — 选几篇笔记，自动整合成结构化周报，不用再凑字数  
**双向笔记** — `[[标题]]` 语法创建链接，自动建立反向引用  
**关系图谱** — 可视化笔记网络，点击跳转，支持缩放拖拽  
**版本历史** — 每次修改都有记录，最多保留 20 个版本，随时可回溯  
**编译真相** — 上线区存放结论，下线区保留原始思考，知识会过时，证据不会  
**离线优先** — 数据存在本地 IndexedDB，完全可控，支持导出 Markdown/HTML/PDF  
**智能搜索** — 全文搜索 + 高级筛选，10000 条笔记 < 0.5 秒
